# Fi Partner Intelligence Tool

AI-powered partnership intelligence tool that researches companies and generates partnership briefs with fit scores, integration angles, and ready-to-send outreach emails.

## Features

- **Company Research**: Uses Claude AI with web search to gather current information about any company
- **Partnership Fit Scoring**: AI-powered scoring (0-100) of partnership fit with Fi
- **Integration Analysis**: Technical and commercial integration path recommendations
- **Outreach Generation**: Ready-to-send cold outreach emails tailored to each company
- **Real-time Processing**: Shows research steps as they happen
